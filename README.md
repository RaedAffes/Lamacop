# LaMaCoP (Frontend)

This repository contains a frontend-only Next.js app under the `frontend/` folder.

## Run

```bash
cd frontend
npm install
npm run dev
```

Open http://localhost:3000

## Build

```bash
cd frontend
npm run build
```

### Content
Update component content in `src/components/` and `src/app/*/page.tsx` files.

## 📱 Responsive Breakpoints

- Mobile: < 768px (md)
- Tablet: 768px - 1024px
- Desktop: > 1024px

## 🔗 Links & Navigation

All pages are interconnected with proper navigation:
- Header navigation menu
- Footer quick links
- Internal page links throughout the site

## 📝 Notes

- This is a frontend-only implementation
- Contact form and login/register are UI only (no backend integration)
- Images/logos are not included - customize the header with actual branding
- All content is placeholder and should be replaced with actual laboratory information
